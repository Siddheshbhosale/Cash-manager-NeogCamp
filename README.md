## About app?
  This is an application to know how many notes of currency 1,5,10,20,100,500,2000 to return to the customer. 
  
## How to use?
   User needs to enter bill to be payed and the amount of cash givrn by the customer, Then click on Check. The App will show the number of notes to return.
